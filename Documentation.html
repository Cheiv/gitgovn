Introduction

This project represents the preferences of an AliExpress application for mobile. It shows the table with chosen preferences that can be chosen in this mobile application. Also this preferences ordered by Alphabet order and have several blocks marked with color. Green color for reminder frequency set on less than 10 days and red for more than 10 days.

Methodology

For making this project I used this technologies

  • XML
  • XML Schema
  • HTML
  • XSL
  • XSLT

All documenst are valid and well formed. Checked using W3C XML Schema (XSD) Validation online

Work content

This project consists of the following files:

  1. Xml file
  2. Xsd file
  3. Xslt file
  4. Html file

Elements description

The root element is AliExpirs_app_settings. It contains 5 different records named Settings. Each of them had different preferences chosen.

Each Settings contains the following:

  • delivery_address
  • currency
    • currency_paying
    • currency_shown
  • language
  • notification_settings
    • notifications
    • Promotions_and_sales
    • mute_notifications
    • seller_masages
    • order_updates
    • order_massages
    • reminder_frequency
      • frequency
  • Theme_settings
    • font
    • theme
    • brightness

Transformation description

XSLT has been used for this transformation of XML to an HTML table with chosen preferences.
